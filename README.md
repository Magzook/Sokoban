# Sokoban
Симулятор кладовщика, MVP. <br>
Здесь должно быть классное описание с кучей эмодзи и ссылками на дискорд

Реализовано: <br>
- три сцены (начальный экран, игровой экран, экран редактора)
- запуск уровня из файла xml
- редактор уровней с сохранением в файл xml

Описание папок проекта: <br>
- .config - для быстрого открытия MCGB Editor'а из Rider'а (мб работает и в других IDE)
- Content - для хранения контента (текстуры, конфигурационные файлы, уровни...). При запуске всё содержимое переносится в bin/Debug/...
- Library - классы-хелперы для UI. Не привязаны к Sokoban'у, можно использовать для построения любых игр с Monogame
- Logic - логика игры (классы склада, уровня)
- Scenes - сцены (главное меню, игровое меню, меню редактора уровней)
- Serialization - сериализация уровня в XML и десериализация из XML
- UI - классы для отрисовки того, что обозначено в Logic
- bin/Debug/net9.0-windows - добавил для демонстрации т.к. там лежит фактически тот контент, который подгружается через Content.RootDirectory
